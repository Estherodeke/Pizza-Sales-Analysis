
# Pizza Sales Analysis

## Project Overview
This project analyzes pizza sales data using Microsoft Excel to answer key business questions for a pizza restaurant.

## Business Questions Answered
1. What days and times do we tend to be busiest?
2. How many pizzas are we making during peak periods?
3. What are our best and worst-selling pizzas?
4. What is our average order value?
5. How well are we utilizing our seating capacity?

## Key Findings
- **Busiest Day:** Friday (8,106 orders)
- **Slowest Day:** Sunday (5,917 orders)
- **Peak Time:** 12PM - 1PM (6,543 orders)
- **Most Popular Pizza:** The Classic Deluxe Pizza (2,453 units sold)
- **Least Popular Pizza:** The Brie Carre Pizza (490 units sold)
- **Average Order Value:** $38.31
- **Table Turnover Rate:** 8.88 times per day

## Tools Used
- Microsoft Excel (Pivot Tables, TEXT, HOUR, IFS functions)

## Dataset
Source: Kaggle - Pizza Sales Dataset
